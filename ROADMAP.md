# Fixtura Roadmap

## Phase 1 — Project Setup

Goal:

Create the foundation.

Tasks:

- Create Expo project
- Configure TypeScript
- Configure Expo Router
- Configure NativeWind
- Configure Zustand
- Create folder structure
- Create base theme

Success:

Application launches successfully.

---

## Phase 2 — Tournament Data

Goal:

Represent World Cup teams and groups.

Tasks:

- Create Team model
- Create Group model
- Add 48 teams
- Add groups
- Add knockout mapping

Success:

Tournament data can be loaded.

---

## Phase 3 — Simulation Engine

Goal:

Create realistic match simulations.

Tasks:

- Team ratings
- Random number generator
- Seed support
- Match simulation
- Goal generation
- Probability balancing

Success:

Simulated matches produce realistic scorelines.

---

## Phase 4 — Group Stage Engine

Goal:

Calculate standings.

Tasks:

- Simulate group matches
- Calculate standings
- Tiebreakers
- Qualification logic

Success:

Groups finish correctly.

---

## Phase 5 — Knockout Engine

Goal:

Generate knockout rounds.

Tasks:

- Extra time
- Penalties
- Champion determination

Success:

Full tournament can be completed.

Goal:
